Console Window Object

Explicação

A propriedade console do objeto Window retorna o objeto Console. Este objeto fornece acesso ao console de depuração do navegador (frequentemente chamado de DevTools ou Ferramentas de Desenvolvedor), onde você pode registrar mensagens, inspecionar valores de variáveis e depurar código.

        
            // Usando a propriedade console do objeto Window
console.log("Esta é uma mensagem de log.");
console.warn("Esta é uma mensagem de aviso.");
console.error("Esta é uma mensagem de erro.");
console.info("Esta é uma mensagem informativa.");
